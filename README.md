# Pokemon App

Android-приложение, в котором отображается список всех покемонов. По нажатию на элемент списка открывается экран с характеристиками выбранного покемона. 

Используемый API: [PokeAPI](https://pokeapi.co)

## Стек

* Kotlin
* Android View
* MVI
* Retrofit
* Kotlin Coroutines
